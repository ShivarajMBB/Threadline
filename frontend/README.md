# Threadline CRM - Demo Setup

## 🚀 Quick Start (60 seconds)

```bash
# 1. Install dependencies
npm install

# 2. Run the app
npm run dev

# 3. Open browser
# App will open at http://localhost:5173
```

## 🎥 Recording Demo Video

1. **Hide mode indicator:** Press `Ctrl + Shift + H`
2. **Fullscreen browser:** Press `F11`
3. **Start recording!**

## ⌨️ Keyboard Shortcuts

- `Ctrl + Shift + M` - Toggle Review/Production Mode
- `Ctrl + Shift + H` - Show/Hide Mode Indicator
- `F11` - Browser Fullscreen
